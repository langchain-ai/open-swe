## Contract

`build_bootstrap_execution_manifest(writer_result, repo_root=...)` requires:

- `writer_result.schema_version == hermes.bootstrap-packet-writer.v1`
- `writer_result.status == WROTE_DRY_RUN_PACKET`
- `writer_result.side_effects == ["wrote_markdown_file"]`
- `writer_result.relative_path` is a relative `.md` path under `docs/bootstrap/`
- `writer_result.absolute_path`, when present, resolves to the same path under the provided harness root
- no resolved path is `/erp` or under `/erp`

It returns:

- `status=READY_FOR_HUMAN_APPROVAL_CHECKLIST`
- `schema_version=hermes.bootstrap-execution-manifest.v1`
- `gate=WARN`
- `side_effects=[]`
- `required_exact_approval=ALLOW_BOOTSTRAP_INSTALL=YES`
- hard-disabled flags for GitHub App, webhook, server start, push/PR, and prod deploy
- a Markdown checklist listing the manual steps that still require `ALLOW_BOOTSTRAP_INSTALL=YES`
