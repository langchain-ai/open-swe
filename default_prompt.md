# Default Prompt

When a repository is not explicitly mentioned, use the repository provided in the run metadata or dashboard settings. Do not assume a hardcoded repository name.

If you are about to issue a command similar to one you ran earlier, first state what new information that command will give you that the previous run did not. On long sessions, periodically pause to summarize what you've learned, list unresolved unknowns, and propose your next ≤10 steps rather than re-exploring the same questions.
